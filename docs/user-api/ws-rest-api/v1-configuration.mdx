The `configuration` endpoint is used to get the session specific configuration of the charging station. The configuration can only change between sessions.

## Values
- `sessionId` - UUID of the charging session.
- `effectiveAmperageLimit` - Absolute amperage limit of the charge point --------- FIXME: description
