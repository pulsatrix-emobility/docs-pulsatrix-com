# <span style={{ display: 'flex', alignItems: 'center' }}>/v1/limits <span style={{ marginLeft: '10px',marginTop: '5px', backgroundColor: '#62B0FE', color: 'white', padding: '4px', borderRadius: '5px', fontSize: '23px' }}>GET</span></span>

the `/limits` endpoint provides controll functionality for the limits of the charger. A `GET` request returns the current values of the limits as set through the API.  


The limits are only active if the `enabled` value is set to `true`.  
If you set the limits for `amperageLimit` and `powerLimit`, the lower effective value will be used.

## Values
- `enabled` - gets or sets the enable state of the limits
- `amperageLimit` - gets or sets the amperage limit
- `powerLimit` - gets or sets the power limit
- `timeout` - gets or sets the data validity timeout in ms

## Example Request


<Tabs>
  <TabItem value="rest" label="REST" default>
    ``` json title="http://127.0.0.1/api/v1/limits"
    {
      "enabled": true,
      "amperageLimit": NaN,
      "powerLimit": NaN,
      "timeout": 300000
    }
    `````
  </TabItem>
  <TabItem value="ws" label="WebSocket">
    ```json title="ws://127.0.0.1:80/api/ws"
    Topic: limits

    {"enabled":false,"amperageLimit":NaN,"powerLimit":NaN,"timeout":300000}
    ```
  </TabItem>
</Tabs>

## <span style={{ display: 'flex', alignItems: 'center' }}>/v1/limits <span style={{ marginLeft: '10px',marginTop: '5px', backgroundColor: '#FCA130', color: 'white', padding: '5px', borderRadius: '5px', fontSize: '23px' }}>SET</span></span>
  The SET endpoint for the limits looks different depending on the protocol used, the REST endpoint allows you to set all values at once, while the WebSocket endpoint only allows you to set one value at a time. 

<Tabs>
  <TabItem value="rest" label="REST" default>
    set the request body to the desired values. If you want to change only one value, you can leave the others out.  
    If you want to give up controll over a value, set it to `NaN`.



    ``` json title="http://127.0.0.1/api/v1/limits"
    {
      "enabled": true,
      "amperageLimit": NaN,
      "powerLimit": NaN,
      "timeout": 300000
    }
    `````
  </TabItem>
  <TabItem value="ws" label="WebSocket">
    - `setEnabled` - sets the enable state of the limits
    - `setCurrentLimit` - sets the amperage limit
    - `setPowerLimit` - sets the power limit
    - `setTimeout` - sets the data validity timeout

    with Websocket it is only possible to set one value at a time.  
    Use the format: `"<command>\n<value>"` to set the value.

    `"setCurrentLimit\n10"` *set the amperage limit to 10A*   
    `"setPowerLimit\n10000"` *sets the Power Limit to 10000W*  
    `"setEnabled\ntrue"` *sets the enable state to true*  
    `"setTimeout\n10000"` *sets the data validity timeout to 10000ms* 

  </TabItem>
</Tabs>
