The fiscal metering topic provides meter data of the EVSE.

- `voltage` - *\[float, float, float\]* - Per phase voltage in V
- `amperage` - *\[float, float, float\]*  - Per phase charging amperage in A
- `frequency` - *float* - Current grid frequency in Hz
- `activePower` - *float* - Current active power in W
- `energyImported` - *float* - Current grid meter value in kWh
